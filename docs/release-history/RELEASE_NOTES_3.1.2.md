# Alcyone 3.1.2

[Скачать Alcyone 3.1.2 для LG webOS](https://raw.githubusercontent.com/ichikanya/Alcyone/main/packages/com.alcyone.vpn_3.1.2_all.ipk)

Обновление полностью перерабатывает главный экран для более понятного и аккуратного управления VPN с телевизора.

### Новый главный экран

- Подключение, состояние VPN и выбранная локация теперь образуют единый визуальный сценарий.
- Основная кнопка и статус хорошо читаются с расстояния и адаптированы для управления пультом.
- Флаг и название сервера отображаются в компактной плашке, которая автоматически подстраивается под короткие и длинные названия.
- Свечение строго центрировано относительно кнопки и плавно растворяется без обрезанных границ.
- Служебный блок веб-импорта сохранён как отдельный второстепенный элемент.

### Проверка

- Проверены варианты с короткими, длинными и предельно длинными названиями серверов.
- Подтверждена корректная геометрия при 1280×720 и отсутствие горизонтального переполнения.
- Пройдены тесты TV UI, жизненного цикла VPN, полного отключения, хранилища и импорта подписок.

Для работы приложения необходим root-доступ. Существующие профили, подписки, настройки и данные в `/var/lib/alcyone` сохраняются при обновлении.
