Alcyone 3.2.0 for LG webOS

Alcyone is a VPN client for rooted LG webOS TVs. It runs VPN directly on the television without requiring a separate router, phone, or computer as a gateway.

Version 3.2.0 is available as two independently installable editions with the same TV-optimized interface, subscription importer, server selection, ping checks, autostart, external-IP checks, logs, Russian/English localization, and D-pad navigation.

XRay Edition

Alcyone XRay keeps the established application identity and all existing data. It uses the current Xray 26.3.27 core with tun2socks and is intended for large subscriptions, many configurations, XHTTP, and complete XRay configs containing routing rules or balancers.

sing-box Edition

Alcyone sing-box uses a separate application identity and data directory. Its sing-box 1.13.14 core is trimmed to the required QUIC and uTLS capabilities and runs one native system-TUN process, avoiding gVisor and optional service/API modules for lower process, descriptor, and startup overhead on low-powered TVs.

Both editions support VLESS, VMess, Trojan, Shadowsocks, SOCKS5, and Hysteria2 links. XHTTP and complete XRay routing/balancer semantics are available in the XRay edition.

Subscriptions and servers can be imported from a phone or computer through the built-in web interface. Compatibility headers and User-Agent profiles help retrieve subscriptions intended for specific VPN clients.

Tunnel logs, idle connections, sockets, descriptors, request buffers, nested subscription work, and background polling are bounded to keep resource use stable.

Root access is required. Both editions can be installed together, but only one TV-wide VPN tunnel should run at a time. When reporting a problem, include the TV model, webOS version, and edition.
