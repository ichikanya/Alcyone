Alcyone 3.1.0

Alcyone is a VPN client for rooted LG webOS TVs. It runs VPN directly on the television through xray and tun2socks, without requiring a separate router, phone, or computer as a gateway.

The application supports VLESS/XHTTP, VMess, Trojan, Shadowsocks, SOCKS5, and Hysteria2 profiles. Subscriptions and servers can be imported from a phone or computer through the built-in web interface.

The Home screen now keeps the selected server focused on its centered flag and name, and Settings subpages use a consistently aligned shared header.

When Alcyone starts or returns to the foreground, it restores the real VPN service state and control interface so an existing tunnel remains visible and can be stopped normally.

Servers can be pinged on demand from the TV and sorted by measured latency or name. Color-coded indicators show good, average, and poor latency, while unavailable servers are marked as n/a.

Compatibility headers and User-Agent profiles can help load subscriptions that are normally available only inside specific VPN clients. Other features include server selection, subscription updates, VPN autostart, external IP checks, country flags, and tunnel logs.

The interface is available in Russian and English and supports full D-pad navigation on LG remotes with or without a pointer.

Root access is required. When reporting a problem, include the TV model and webOS version.
