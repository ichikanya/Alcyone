Alcyone 3.1.7

Alcyone is a VPN client for rooted LG webOS TVs. It runs VPN directly on the television through xray and tun2socks, without requiring a separate router, phone, or computer as a gateway.

The application supports VLESS/XHTTP, VMess, Trojan, Shadowsocks, SOCKS5, and Hysteria2 profiles. Subscriptions and servers can be imported from a phone or computer through the built-in web interface.

The Home screen uses a TV-optimized connection layout with a prominent VPN control, clear connection state, centered glow, and a compact selected-location badge that adapts to short and long server names. Server-list navigation preserves the current position and scrolls only as far as needed to reveal a focused server.

When Alcyone starts or returns to the foreground, it restores the real VPN service state. Disconnecting fully removes VPN processes and TUN network state, then restarts the local web control interface on the restored connection.

Version 3.1.7 prevents gradual Xray descriptor exhaustion by multiplexing XHTTP requests, expiring idle and half-closed connections sooner, and giving only Xray bounded additional descriptor headroom. This prevents repeated too many open files failures from slowing down and eventually freezing the TV.

Selecting a different server while VPN is active now performs a complete disconnect and reconnect with the new profile. The log viewer can physically clear all managed log files and reports current descriptor use for Xray, tun2socks, and the web service.

Tunnel logs, idle connections, sockets, descriptors, request buffers, nested subscription work, and background polling are bounded to keep memory, CPU, and storage use stable on resource-constrained TVs.

Servers can be pinged on demand from the TV and sorted by measured latency or name. Color-coded indicators show good, average, and poor latency, while unavailable servers are marked as n/a.

Compatibility headers and User-Agent profiles can help load subscriptions that are normally available only inside specific VPN clients. Other features include server selection, subscription updates, VPN autostart, external IP checks, country flags, and tunnel logs.

The interface is available in Russian and English and supports full D-pad navigation on LG remotes with or without a pointer.

Root access is required. When reporting a problem, include the TV model and webOS version.
